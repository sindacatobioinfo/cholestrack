users - to create and manage accounts. This app will allow the account creation and login
profile - the app to manage user profile. So once the user creates an account, the view redirects to this other app, to make the user create a profile with further information (name, team, role, institutional mail address)
home - is the page which the user sees when he/she logs in or after finish the profile creation. In this page, the user will have access to their profile (for any edition), and available tools (for now, only sample management will be available).
samples - this one will allow the user to add and view sample data already inserted to the database. The info displayed is that found in the actual "home" page.
files - handle with sample data files
